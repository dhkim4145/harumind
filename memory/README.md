\# 하루마음 – 같은 그림 찾기 (인지 자극 두뇌놀이)



하루마음은 누구나 쉽게 사용할 수 있는  

웹 기반 인지 자극 · 기억력 향상 두뇌 놀이 서비스입니다.



---



\## 📁 프로젝트 구조



각 파일은 역할 기준으로 분리되어 있습니다.



\- \*\*index.html\*\*  

&nbsp; → 레이아웃 및 마크업 전용  

&nbsp; (화면 구조, 버튼, 카드 배치 등)



\- \*\*ui.js\*\*  

&nbsp; → 화면·상태·사운드·UX 로직  

&nbsp; (메시지 표시, 효과음, 배경음악, 팝업, 토글 UI 등)



\- \*\*game.js\*\*  

&nbsp; → 게임 규칙 및 로직  

&nbsp; (카드 생성, 매칭 판정, 점수 계산, 게임 진행)



\- \*\*storage.js\*\*  

&nbsp; → 데이터 저장  

&nbsp; (오늘 기록, 최고 점수, 설정값을 localStorage에 저장)



\- \*\*config.js\*\*  

&nbsp; → 공통 설정값  

&nbsp; (키, 기본 옵션, 상수 정의)



---



\## 🎯 설계 원칙



\- 역할별 파일 분리 (Separation of Concerns)

\- 기능 추가 시 기존 코드 영향 최소화

\- 모바일·어르신 사용성을 우선 고려한 UX

\- 별도 회원가입 없이 기기 내 저장 방식 사용
